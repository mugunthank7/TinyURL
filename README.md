# TinyURL Project

This is a simple URL shortening service similar to TinyURL.

## Features

- Shorten long URLs
- Redirect to original URLs using shortened versions
- Basic web interface for URL input

## Setup

1. Clone the repository:
    ```
    git clone https://github.com/yourusername/TinyURL-Project.git
    ```

2. Install the dependencies:
    ```
    pip install -r requirements.txt
    ```

3. Run the application:
    ```
    python app.py
    ```

## Usage

Open your web browser and go to `http://127.0.0.1:5000/`.
